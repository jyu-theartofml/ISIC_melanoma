I implemented the Local Interpretable Model-Agnostic Explanations(LIME) algorithm to display the features from the image that had noticeable influence on the predicted outcome. The figure below shows the output of LIME along with the original image, where the green superpixels define regions that support the prediction, and red is against the prediction. Due to the setup of LIME, this was done using a DenseNet model trained without meta data.

There's also a state-of-the-art method called SHapley Additive exPlanation (SHAP) that applies Shapley value for estimating feature influence in a model. It's a much more computationally intensive approach because it looks for the average marginal contribution of a feature across the possible predictions, therefore assigning contribution of the features with respect to the prediction. The original paper shows that it guarantees consistency and local accuracy. In contrast, LIME runs faster but it only provides an explanation that shows the most important attributes for a given input.
